# FoE-Tab-Tool-v2.0
A small browser extension to rename FoE tabs, give emoji notifications for finished productions and messages, and provide quick links to data sites for each world!

## Features
1. Renames tabs from **Forge of Empires** to **worldID | worldName**, for example "**us26 | Angkor**
2. Adds emojis to the tab name for certain conditions:
    - ♻️ for a warning to refresh the tab (if open for >4 hours)
    - 📦 for finished productions
    - 🔔 for new messages in the message center (excluding official announcements)
    - These emojis can be toggled on or off on a per-tab basis via clicking on the extension
3. Clicking on the extension gives some quick links to popular FoE data websites (not affiliated)

## Installation
The easiest way to install is via the [Chrome Web Store](https://chromewebstore.google.com/detail/foe-tab-tool/genljbnikgboknenccecbknmlfmpgdij), but if you want to manually install the extension, follow the below steps:
1. Download the `FoE Tab Renamer v2.0.3.zip` file
2. Extract all the files to your desired location
3. Head to `chrome://extensions` and click "Load Unpacked"
4. Choose the folder that you extracted the files to
5. It's now installed and ready to go!
